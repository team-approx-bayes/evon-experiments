# Source Code Documentation

This directory contains the source code for the `vonsoap` package and related scripts.

## Directory Structure

*   **`vonsoap/`**: The main Python package.
    *   `optimizers/`: Contains the optimizer implementations (SOAP, EVON, IVON).
*   **`scripts/`**: Executable scripts for experiments and visualizations.
    *   `interactive_2d_comparison.py`: Visualizes optimization trajectories and posterior approximations on a 2D toy problem.
    *   `compare_ivon.py`: Compares different optimizers (e.g., IVON vs IVON2) on MNIST.
